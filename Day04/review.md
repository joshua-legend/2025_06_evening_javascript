variable[변수]

- 데이터 담는 공간

datatype[데이터타입]

- 기본타입: 숫자(number), 문자(string), 불리언(boolean), null, undefined
- 참조타입:

typecasting[타입 바꾸기]

- 문자화: String(), "" + 어떤타입(연결연산자)
- 숫자화: Number(), +(단항연산자), parseInt parseFloat
- 불리언화: Boolean(), !, 비교연산자

operator[연산자]

- 산술연산자
- 대입연산자 =
- 비교연산자 ==,!=, >,<
- 논리연산자 &&,||,!
- 삼항연산자 ?

conditional statement[조건문]

- 옵셔널한 조건문: if(조건식), else if \* n, else
